# Welcome to Telangana

A visually rich landing page dedicated to the state of Telangana, built entirely using HTML and CSS.

## Features

- Hero section with Telangana-themed background
- Custom typography using local fonts
- About Telangana section
- Pillars of Identity cards
- State statistics
- Festivals and Literature sections
- Tourist destinations showcase
- Inspirational Telangana quote section
- Footer with state information

## Technologies Used

- HTML5
- CSS3
- Custom Fonts (@font-face)
